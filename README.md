# Mental Health & Depression in Students – Data Mining Project

This project analyzes student mental health data in order to identify key factors associated with depression.

The goal is to apply data mining and machine learning techniques to explore patterns and predict depression risk among students.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Dataset

The dataset contains approximately **500 student records** including:

- Gender
- Sleep hours
- Academic pressure
- Financial stress
- Dietary habits
- Suicidal thoughts

---

## Methods

### Exploratory Data Analysis (EDA)

Visualization of patterns between depression and behavioral indicators.

### Supervised Machine Learning

Models used:

- Logistic Regression
- Random Forest

Both models achieved approximately **97% accuracy**.

### Unsupervised Learning

Clustering using:

- K-Means
- PCA (Principal Component Analysis)

---

## Key Insights

Depression among students is strongly correlated with:

- academic pressure
- suicidal thoughts
- financial stress
- poor sleep habits

Machine learning models show strong predictive performance and could support early intervention in student mental health.
